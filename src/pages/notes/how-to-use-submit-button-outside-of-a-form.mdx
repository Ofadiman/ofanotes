<MDXIntro>
  The HTML `form` element represents a document section containing interactive
  controls for submitting information.
</MDXIntro>

### The `form` attribute

To submit a form when the button isn't a child of the parent form, you can use the `form` attribute.

```html
<form id="myForm">
  <label for="email">Email:</label>
  <input type="email" name="email" placeholder="Email" />
</form>

<button type="submit" form="myForm">Submit!</button>
```
