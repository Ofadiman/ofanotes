# Webstorm shortcuts

If you are looking for JetBrains IDE shortcuts to increase productivity, that's a good place to start.

---

0. `Alt + 1` - project (go to project tree)
0. `Alt + 2` - git (open version control system window)
0. `Alt + 3` - view docker services
0. `Alt + A` - error description
0. `Alt + B` - branches (git)
0. `Alt + E` - fix eslint problems
0. `Alt + H` - jump to navigation bar
0. `Alt + I` - rearrange code
0. `Alt + K` - unselect occurrence
0. `Alt + M` - add selection for next occurrence
0. `Alt + O` - optimize imports
0. `Alt + Q` - surround with
0. `Alt + R` - rename (file, folder or code occurrence)
0. `Alt + S` - sort lines
0. `Alt + U` - toggle case (string)
0. `Alt + V` - vcs operations popup
0. `Alt + X` - reformat with prettier
0. `Alt + Z` - soft wrap
0. `Alt + Button 1 click` - add or remove caret
0. `Alt + Delete` - safe delete
0. `Alt + Enter` - show context actions (quick fix)
0. `Alt + Insert` - generate (code)
0. `Alt + [` - go to previous splitter
0. `Alt + ]` - go to next splitter
0. `Alt + ←` - move caret to line start
0. `Alt + ↑` - move caret to line end with selection
0. `Alt + →` - move caret to line end
0. `Alt + ↓` - move caret to line start with selection
0. `Ctrl + B` - go to declaration or usages
0. `Ctrl + D` - duplicate line or selection
0. `Ctrl + E` - forward (navigate to next caret position)
0. `Ctrl + F` - find (current file)
0. `Ctrl + G` - fetch (git)
0. `Ctrl + H` - compare with the same repository version
0. `Ctrl + J` - pull (git)
0. `Ctrl + K` - commit (git)
0. `Ctrl + Q` - quick documentation
0. `Ctrl + R` - replace
0. `Ctrl + S` - save all
0. `Ctrl + T` - quick type definition
0. `Ctrl + W` - back (navigate to previous caret position)
0. `Ctrl + Z` - undo
0. `Ctrl + +` - expand (code block)
0. `Ctrl + -` - collapse (code block)
0. `Ctrl + /` - comment with line comment
0. `Ctrl + Backspace` - delete to word start
0. `Ctrl + Delete` - delete to word end
0. `Ctrl + Space` - basic (code completion)
0. `Ctrl + Tab` - select next tab
0. `Ctrl + [` - delete to line start
0. `Ctrl + ]` - delete to line end
0. `Ctrl + ↑` - move up and scroll
0. `Ctrl + ↓` - move down and scroll
0. `Ctrl + Alt + J` - javascript file
0. `Ctrl + Alt + M` - select all occurrences
0. `Ctrl + Alt + N` - create new directory or package
0. `Ctrl + Alt + P` - file (create a new scratch file)
0. `Ctrl + Alt + S` - save as live template
0. `Ctrl + Alt + T` - typescript file
0. `Ctrl + Alt + V` - paste from history
0. `Ctrl + Alt + Z` - rollback (git)
0. `Ctrl + Alt + [` - unsplit
0. `Ctrl + Alt + ]` - split and move right
0. `Ctrl + Shift + C` - close others (editor tabs)
0. `Ctrl + Shift + C` - copy (terminal)
0. `Ctrl + Shift + F` - find in path
0. `Ctrl + Shift + K` - push (git)
0. `Ctrl + Shift + N` - go to file
0. `Ctrl + Shift + S` - go to symbol
0. `Ctrl + Shift + T` - go to test
0. `Ctrl + Shift + V` - close (editor tabs)
0. `Ctrl + Shift + V` - paste (terminal)
0. `Ctrl + Shift + X` - close all (editor tabs)
0. `Ctrl + Shift + Z` - redo
0. `Ctrl + Shift + +` - expand all (code blocks)
0. `Ctrl + Shift + -` - collapse all (code blocks)
0. `Ctrl + Shift + /` - comment with block comment
0. `Ctrl + Shift + Tab` - select previous tab
0. `Ctrl + Shift + F10` - run context configuration
0. `Shift (press x2)` - search everywhere
0. `Shift + Alt + Enter` - start new line before current
0. `Shift + Alt + ←` - shrink selection
0. `Shift + Alt + ↑` - move line up
0. `Shift + Alt + ↑` - move statement up
0. `Shift + Alt + →` - expand selection
0. `Shift + Alt + ↓` - move line down
0. `Shift + Alt + ↓` - move statement down
0. `Shift + Enter` - start new line
0. `Shift + F1` - run (current context)
0. `Shift + F2` - stop (current context)
0. `F5` - reload all from disk
0. `F6` - restart typescript service
0. `F8` - show usages
0. `F9` - settings
0. `F11` - toggle bookmark
0. `F12` - show bookmarks
