### Webstorm shortcuts

0. `Alt + 1` - project (go to project tree)
1. `Alt + 2` - git (open version control system window)
2. `Alt + 3` - view docker services
3. `Alt + A` - error description
4. `Alt + B` - branches (git)
5. `Alt + E` - fix eslint problems
6. `Alt + H` - jump to navigation bar
7. `Alt + I` - rearrange code
8. `Alt + K` - unselect occurrence
9. `Alt + M` - add selection for next occurrence
10. `Alt + O` - optimize imports
11. `Alt + Q` - surround with
12. `Alt + R` - rename (file, folder or code occurrence)
13. `Alt + S` - sort lines
14. `Alt + U` - toggle case (string)
15. `Alt + V` - vcs operations popup
16. `Alt + X` - reformat with prettier
17. `Alt + Z` - soft wrap
18. `Alt + Button 1 click` - add or remove caret
19. `Alt + Delete` - safe delete
20. `Alt + Enter` - show context actions (quick fix)
21. `Alt + Insert` - generate (code)
22. `Alt + [` - go to previous splitter
23. `Alt + ]` - go to next splitter
24. `Alt + ←` - move caret to line start
25. `Alt + ↑` - move caret to line end with selection
26. `Alt + →` - move caret to line end
27. `Alt + ↓` - move caret to line start with selection
28. `Ctrl + B` - go to declaration or usages
29. `Ctrl + D` - duplicate line or selection
30. `Ctrl + E` - forward (navigate to next caret position)
31. `Ctrl + F` - find (current file)
32. `Ctrl + G` - fetch (git)
33. `Ctrl + H` - compare with the same repository version
34. `Ctrl + J` - pull (git)
35. `Ctrl + K` - commit (git)
36. `Ctrl + Q` - quick documentation
37. `Ctrl + R` - replace
38. `Ctrl + S` - save all
39. `Ctrl + T` - quick type definition
40. `Ctrl + W` - back (navigate to previous caret position)
41. `Ctrl + Z` - undo
42. `Ctrl + +` - expand (code block)
43. `Ctrl + -` - collapse (code block)
44. `Ctrl + /` - comment with line comment
45. `Ctrl + Backspace` - delete to word start
46. `Ctrl + Button 1 Click` - go to declaration or usages
47. `Ctrl + Button 3 Click` - highlight current scope
48. `Ctrl + Delete` - delete to word end
49. `Ctrl + Space` - basic (code completion)
50. `Ctrl + Tab` - select next tab
51. `Ctrl + [` - delete to line start
52. `Ctrl + ]` - delete to line end
53. `Ctrl + ↑` - move up and scroll
54. `Ctrl + ↓` - move down and scroll
55. `Ctrl + Alt + J` - javascript file
56. `Ctrl + Alt + M` - select all occurrences
57. `Ctrl + Alt + N` - create new directory or package
58. `Ctrl + Alt + P` - file (create a new scratch file)
59. `Ctrl + Alt + S` - save as live template
60. `Ctrl + Alt + T` - typescript file
61. `Ctrl + Alt + V` - paste from history
62. `Ctrl + Alt + Z` - rollback (git)
63. `Ctrl + Alt + [` - unsplit
64. `Ctrl + Alt + ;` - emoji & symbols
64. `Ctrl + Alt + ]` - split and move right
65. `Ctrl + Shift + C` - close others (editor tabs)
66. `Ctrl + Shift + C` - copy (terminal)
67. `Ctrl + Shift + F` - find in path
68. `Ctrl + Shift + K` - push (git)
69. `Ctrl + Shift + N` - go to file
70. `Ctrl + Shift + S` - go to symbol
71. `Ctrl + Shift + T` - go to test
72. `Ctrl + Shift + V` - close (editor tabs)
73. `Ctrl + Shift + V` - paste (terminal)
74. `Ctrl + Shift + X` - close all (editor tabs)
75. `Ctrl + Shift + Z` - redo
76. `Ctrl + Shift + +` - expand all (code blocks)
77. `Ctrl + Shift + -` - collapse all (code blocks)
78. `Ctrl + Shift + /` - comment with block comment
79. `Ctrl + Shift + Tab` - select previous tab
80. `Ctrl + Shift + F10` - run context configuration
81. `Shift (press x2)` - search everywhere
82. `Shift + Alt + Enter` - start new line before current
83. `Shift + Alt + ←` - shrink selection
84. `Shift + Alt + ↑` - move line up
85. `Shift + Alt + ↑` - move statement up
86. `Shift + Alt + →` - expand selection
87. `Shift + Alt + ↓` - move line down
88. `Shift + Alt + ↓` - move statement down
89. `Shift + Enter` - start new line
90. `Shift + F1` - run (current context)
91. `Shift + F2` - stop (current context)
92. `F5` - reload all from disk
93. `F6` - restart typescript service
94. `F8` - show usages
95. `F9` - settings
96. `F11` - toggle bookmark
97. `F12` - show bookmarks
